
# Gestionnaire de Contacts - Projet Java
# Developpé par SEKA ACHI FRANCK. Etudiant a PIGIER en Licence pro
Ce projet est un exercice pratique réalisé dans le cadre du **Chapitre 06 : Programmation Orientée Objet (POO)**. Il permet de gérer une liste de contacts en utilisant des classes, des objets et des méthodes de recherche.

## Objectifs du projet
- Créer et structurer des classes Java.
- Maîtriser l'instanciation d'objets.
- Utiliser des tableaux pour stocker des objets.
- Implémenter des méthodes pour ajouter et rechercher des données.

## Structure du code
Le projet repose sur deux classes principales :
1. **`Contact`** : Modèle de données contenant `name` et `phoneNumber`.
2. **`Main`** : Contient la logique du gestionnaire, le stockage (tableau) et la méthode `main` pour exécuter le programme.

## Prérequis
- **JDK 21** ou supérieur.
- **IntelliJ IDEA** (ou tout autre IDE supportant Java).

## Instructions d'exécution
1. Clonez ce dépôt sur votre machine : `git clone <URL_DE_VOTRE_DEPOT>`
2. Ouvrez le dossier dans **IntelliJ IDEA**.
3. Assurez-vous que le SDK du projet est configuré sur le JDK 21.
4. Exécutez la classe `Main.java`.

## Exemple d'utilisation
Le programme initialise 5 contacts, les stocke dans le gestionnaire, puis effectue une recherche pour afficher le numéro d'un contact spécifique ("Charlie").
